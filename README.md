# MindfulnessPlatform 

## 1. Project verview and Vision
Mindefulness AI Platform은 다면적 웰빙 생태계를 위한 spring Boot3 (Java 21) foundation 입니다. 
이 서비스는 콘텐츠, 개인화 추천, 파트너 통합을 조율하며 마음챙김 경험을 제공합니다. 
우리의 비전은 "마음챙김의 보편화"아며 사용자, 마음챙김 크리에이터, 직원들의 resilence에 관심이 있는 organization사이를 연결하는 
중재자 커뮤니티로 기여합니다. 
까다롭게 설계된 웰니스 여정과 적응형 AI를 결합하여 세대와 국경을 넘어 마음챙김 실천의 장벽을 낮추고 장기적인 습관 형성을 촉진합시다.

## 2. BM과 platform 전략
- **마켓플레이스 플라이 휠**: 마음 챙김을 찾는 유저와 매칭, 구독 등급, 단품세션, 큐레이션 번들로 수익화 합니다. 
- **크리에이터 지원**: 수익분배 대시보드, 성과 분석, 콘텐츠 도구를 제공해 웰니스 크리에이터 생태계를 확장합니다. 

## 3. 시스템 아키텍쳐와 확장성
- 도메인 중심 서비스: 사용자 프로필, 세션, 콘텐츠 카탈로그, 추천, 결제를 위한 모듈형 spring boot요소가 비동기 메세징으로 통신합니다. 
- 이벤트 우선 설계: Kafka이벤트 스트립이 마음챙김 상호작용을 수집해 분석, 습관 추적, 추적 학습을 지원합니다. 
- 탄력 인프라: 컨테이너호된 배보와 자동확장
- 데이터 영속성: 트랜잭션 데이터는 postgresql, 저지연 세션 생태는..
